1.1 Що таке HTML?

HTML означає Hyper Text Markup Language (мова гіпертекстової розмітки) Це стандартизована мова розмітки документів для перегляду вебсторінок у браузері.

1.4. HTML Елементи

HTML елемент зазвичай складається із тегу початкового і тегу кінцевого, а зміст вставляється поміж ними.HTML елементи без змісту називаються порожніми елементами. Порожні елементи не мають кінцевого тегу, наприклад, як елемент (який вказує на розрив рядка).

Елемент "body" визначає тіло документа.

1.5. HTML Атрибути

Атрибут href HTML посилання визначаються за допомогою тега "a". Адреса посилання вказується в атрибуті href

Це посилання

Атрибут src HTML зображення визначаються за допомогою тега "img". Ім'я файла джерела зображення вказується в атрибуті src

Атрибут alt вказує альтернативний текст, який буде використовуватись, якщо зображення не може бути відображене.

Атрибут style використовується для вказівки стилю елемента, такого як колір, шрифт, розмір і т.д.

Це параграф червоного кольору.

Атрибут lang Мова документу може бути оголошена в тезі html. Мова оголошується за допомогою атрибута lang

1.6. HTML Параграфи

HTML елемент pre визначає форматований текст. Текст всередині елементу pre відображається шрифтом фіксованої ширини (зазвичай Courier), і він зберігає пробіли та розриви рядків

1.8. HTML Зображення

1.8.1. HTML Карти зображень

Ідея карти зображення полягає в тому, що ви повинні мати можливість виконувати різні дії залежно від того, де на зображенні ви клацаєте. Для створення карти зображень вам потрібно зображення та деякий HTML-код, який описує області, на які можна натиснути.Зображення вставляється за допомогою тегу img Єдина відмінність від інших зображень полягає в тому, що ви повинні додати атрибут usemap. Значення usemap починається з хеш-тегу # за яким слідує ім'я карти зображення і використовується створити взаємозв'язок між зображенням та картою зображень.
